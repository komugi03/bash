[スクリプトファイル名]
icalc

[パラメータ]
なし

[標準出力]
左項値と右項値を演算した値を以下の形式で出力する。
<左項値> <演算子> <右項値> = <演算結果>
例）100 + 200 = 300

[標準エラー出力]
不正な演算子が入力された場合には、標準エラー出力に「エラー:」から始まるエラー文字列を出力する。
その後、再度演算子の入力を求める。

不正な整数値が入力された場合には、標準エラー出力に「エラー:」から始まるエラー文字列を出力する。
その後、再度整数値の入力を求める。

不正なyes/no値が入力された場合には、標準エラー出力に「エラー:」から始まるエラー文字列を出力する。
その後、再度yes/no値の入力を求める。

ゼロ除算の場合には、標準エラー出力に「エラー:」から始まるエラー文字列を出力する。

[注意点]
入力プロンプトは echo ではなく read -p で出力すること。

[リターンコード]
正常終了時: 0

[実行例]
$ ./icalc
演算子を入力してください (+ - * /): ?
エラー: 不正な演算子が入力されました。
演算子を入力してください (+ - * /): +
左項値を入力してください (整数): X
エラー: 整数以外の値が入力されました。
左項値を入力してください (整数): 100
右項値を入力してください (整数): Y
エラー: 整数以外の値が入力されました。
右項値を入力してください (整数): 200
100 + 200 = 300
処理を継続しますか？ (yes/no): yes
演算子を入力してください (+ - * /): /
左項値を入力してください (整数): 100
右項値を入力してください (整数): 0
エラー: ゼロで除算しようとしました。
処理を継続しますか？ (yes/no): no
処理を終了します。

$ echo $?
0
